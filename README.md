# Dannyca’s Kitchen

A full-featured food ordering website with collection/delivery options and Stripe payments.

Built with Next.js 15, Tailwind, Prisma + PostgreSQL (ready for Vercel Postgres or Supabase), and Stripe.

## Features
- Menu with admin add/edit items
- Cart
- Collection or Delivery (5km radius)
- Secure Stripe checkout
- Simple admin panel (password protected)
- Responsive design
- English first (Philippines version later)

## Deployment
Deploy on Vercel and add environment variables.
